# Fun with opencv face detector using PC camera

```
 $ pip install requirements.txt
 $ python ./videoface.py
```

## Usage
```
 $ ./videoface.py -h
usage: videoface.py [-h] [-b BLUR] [-e] [-s]

videofacce

optional arguments:
  -h, --help            show this help message and exit
  -b BLUR, --blur BLUR  blur N faces
  -e, --eyes            detect eyes
  -s, --smiles          detect smiles
```
